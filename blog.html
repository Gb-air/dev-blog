Fixing the Guestbook Security Flaw
Posted by Cyber_Coder @ 08:45 PM | Category: PHP & MySQL | Comments (3)

Yesterday someone posted an XSS script into my guestbook form and caused an infinite alert loop for everyone visiting. Oops! Note to self: never trust user input.

I fixed it by forcing the text input through the PHP function below before parsing it into the database table. Tables don't break anymore!

<?php
// Prevent scripts injection in Y2K
$clean_name = htmlspecialchars($_GET['name']);
$clean_comment = strip_tags($_GET['comment']);
echo "Saved entry for: " . $clean_name;
?>

Why CSS Layouts are a Scam
Posted by Cyber_Coder @ 02:10 AM | Category: HTML / CSS | Comments (14)

Some web design purists on Slashdot keep telling me to stop using HTML tables for structural layout and switch entirely to CSS div containers and float: left; rules.

I spent 5 hours trying to make a basic 2-column sidebar layout using divs today. It looks fine in Internet Explorer 5, but completely breaks in Netscape Navigator 4.7! The sidebar falls all the way to the bottom of the screen.

Conclusion: Tables are standard, robust, and they work everywhere. CSS is just a passing trend for text color and links.
